Chapter 1. Introduction

 

1.1 Variants of the linear programming problem

The general linear programming problem is

minimizecTx=∑i=1ncixisubject toaiTx≥bi,i∈M1,aiTx≤bi,i∈M2,aiTx=bi,i∈M3,xj≥0,j∈N1,xj≤0,j∈N2,xj free,j∈N3,

where {M1,M2,M3} is a partition of {1,…,m} and {N1,N2,N3} is a partition of {1,…,n}. (Notice that the constraint xj=0 has no effect on the object function.)

  • Cost vector: c=(c1,…,cn)∈Rn

  • Decision variable: x=(x1,…,xn)∈Rn

  • Feasible solution: x∈Rn​ satisfying all of the constraints

  • Feasible set: The set of all feasible solutions

  • Free (unrestricted) variable: xj for j∈N3, i.e., there is no restriction on xj

  • Objective function: x∈Rn↦cTx=∑icixi∈R​

A feasible solution x∗ that minimizes the objective function is called an optimal (feasible) solution.

  • The value of cTx∗∈R is called the optimal cost.

  • For any K∈R, if we can find a feasible solution x such that cTx<K, we say that the optimal cost is −∞ or that the cost is unbounded below.

No need to study maximization problems: Maximizing cTx is equivalent to minimizing −cTx​.

 

General form

A linear programming problem in general form is

minimizecTxsubject toAx≥b,

where b=(b1,…,bm)∈Rm and A is the m×n matrix whose rows are a1T,…,amT.

  1. An equality constraint aiTx=bi is equivalent to the two constraints aiTx≥bi and aiTx≤bi.

  2. Constraints of the form aiTx≤bi can be rewritten as (−ai)Tx≥−bi​.

  3. Constraints of the form xj≥0 or xj≤0 are special cases of constraints of the form aiTx≥bi, where ai=±1j=(0,…,±1,…,0) and bi=0.

We say that two problems are equivalent if, given a feasible solution to one problem, we can construct a feasible solution to the other, with the same cost.

In particular, two equivalent problems have the same optimal cost, and given an optimal solution to one problem, we can construct an optimal solution to the other.

 

Standard form

A linear programming problem in standard form is

minimizecTxsubject toAx=bx≥0

A general linear programming problem can be transformed into an equivalent problem in standard form.

  1. Elimination of free variables: Any real number can be written as the difference of two nonnegative real numbers. Thus, we replace a free (unrestricted) variable xj by xj+−xj−, where xj+ and xj− are new variables with the sign constraints xj+≥0 and xj−≥0.

  2. Elimination of inequality constraints: An inequality constraint aiTx≤bi is equivalent to the standard form aiTx+si=bi and si≥0. Such a (new) variable si is called a slack variable. Similarly, aiTx≥bi is equivalent to aiTx−si=bi and si≥0. In this case, si is called a surplus variable.
