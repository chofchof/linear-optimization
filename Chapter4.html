Chapter 4. Duality Theory

 

4.1 Motivation

We associate a price variable (or Lagrange multiplier) with each constraint and start searching for prices under which the presence or absence of the constraints does not affect the optimal cost.

A. The primal problem in the standard form

Consider the standard form problem

minimizecTxsubject toAx=bx≥0,

which we call the primal problem, and assume that an optimal solution x∗ exists.

We introduce a relaxed problem with a price vector p∈Rm​

minimizecTx+pT(b−Ax)subject tox≥0.

Then the optimal cost g(p) of the relaxed problem for each p is

g(p)=minx≥0[cTx+pT(b−Ax)]≤cTx∗+pT(b−Ax∗)=cTx∗,

since x∗ is feasible, so that g(p) is a lower bound for the optimal cost cTx∗. Moreover,

g(p)=pTb+minx≥0(cT−pTA)x=pTb+{0,if cT−pTA≥0T−∞,otherwise.

The dual problem searches for the tightest possible lower bound, maxp∈Rmg(p).

maximizepTbsubject topTA≤cT.

B. The primal problem in the general form

Consider the primal in the general form

minimizecTxsubject toAx≥b,

which is equivalent to the problem in the standard form

minimizecTxsubject toAx−s=bs≥0.

Since the equality Ax−s=b can be written in the form [A|−Im][xs]=b, the dual constraints become pT[A|−Im]≤[cT|0mT] or pTA≤cT,p≥0. However, in this case s≥0 but x is free.

Actually, the optimal cost for the relaxed problem is

g(p)=minx∈Rn,s≥0[cTx+pT(b−Ax+s)]=pTb+minx∈Rn(cT−pTA)x+mins≥0pTs=pTb+{0,if cT−pTA=0T−∞,otherwise+{0,if p≥0−∞,otherwise.

Therefore, the dual problem is

maximizepTbsubject topTA=cTp≥0.

Notice that if the equality is replaced to Ax+s=b (in the case of Ax≤b), then the dual constraints should be replaced to p≤0.

 

4.2 The dual problem

Given a primal problem with the structure shown on the left, its dual is defined to be the maximization problem shown on the right:

minimizecTxmaximizepTbsubject toaiTx≥bi,subject topi≥0,i∈M1,aiTx≤bi,pi≤0,i∈M2,aiTx=bi,pi free,i∈M3,xj≥0,pTAj≤cjj∈N1,xj≤0,pTAj≥cjj∈N2,xj free,pTAj=cjj∈N3.

Theorem 4.1 If we transform the dual into an equivalent minimization problem and then form its dual, we obtain a problem equivalent to the original problem. In other words, "the dual of the dual is the primal."

Theorem 4.2 Suppose that we have transformed a linear programming problem Π1 to another linear programming problem Π2, by a sequence of transformations of the following types:

  1. Replace a free variable with the difference of two nonnegative variables.

  2. Replace an inequality constraint by an equality constraint involving a nonnegative slack variable.

  3. If some row of A in a feasible standard form problem is a linear combination of the other rows, eliminate the corresponding equality constraint.

Then, the duals of Π1 and Π2 are equivalent, i.e., they are either both infeasible, or they have the same optimal cost.

 

4.3 The duality theorem

A primal and its dual can be stated more compactly, in matrix notation. In standard form,

minimizecTxmaximizepTbsubject toAx=bsubject topTA≤cT,x≥0,

and in general form,

minimizecTxmaximizepTbsubject toAx≥b,subject topTA=cTp≥0.

For x∈Rn and p∈Rm, we define

ui≡pi(aiTx−bi)andvj≡(cj−pTAj)xj.

Since ∑iui=pT(Ax−b) and ∑jvj=(cT−pTA)x​, we have

∑iui+∑jvj=cTx−pTb.

Theorem 4.3 (Weak duality) If x is a feasible solution to the primal problem and p is a feasible solution to the dual problem, then pTb≤cTx​.

Proof. If x and p are primal and dual feasible, respectively, then ui=0 and vj≥0 in standard form, and ui≥0 and vj=0 in general form. In both cases, we have ∑iui+∑jvj=cTx−pTb≥0.

Corollary 4.1 (a) If the optimal cost in the primal is −∞, then the dual problem must be infeasible. (b) If the optimal cost in the dual is ∞, then the primal problem must be infeasible.

Corollary 4.2 Let x and p be feasible solutions to the primal and the dual, respectively. If pTb=cTx, then x and p are optimal solutions to the primal and the dual, respectively.

Theorem 4.4 (Strong duality) If a linear programming problem has an optimal solution, then so does its dual, and the respective optimal costs are equal.

Proof. (Case 1) Consider the standard form problem: Minimize cTx subject to Ax=b and x≥0.

  1. Assume that the rows of A are linearly independent and that there exists an optimal solution.

  2. Apply the simplex method to this problem. As long as cycling is avoided, the simplex method terminates with an optimal solution x∗ and an optimal basis B.

  3. When the simplex method terminates, the reduced costs must be nonnegative, i.e., c¯T=cT−cBTB−1A≥0T.

  4. Define pT≡cBTB−1​. Then pTA≤cT, which shows that p is a feasible solution to the dual problem: Maximize pTb subject to pTA≤cT.

  5. In addition, pTb=cBTB−1b=cBTxB=cTx∗, since x∗​ is a basic feasible solution.

  6. By Corollary 4.2, p​ is an optimal solution to the dual, and the optimal dual cost is equal to the optimal primal cost.

(Case 2) Consider the general form problem Π1 that has an optimal solution. We first transform Π1 into an equivalent standard form problem Π2 with the same optimal cost, and in which the rows of A are linearly independent.

In a linear programming problem, exactly one of the following three possibilities will occur:

  1. (Finite optimum) There is an optimal solution.

  2. (Unbounded) The problem is unbounded; that is, the optimal cost is −∞ (for primal problems) or ∞ (for dual problems).

  3. (Infeasible) The problem is infeasible.

This leads to nine possible combinations for the primal and the dual.

 

Complementary slackness

Theorem 4.5 (Complementary slackness) Let x and p be feasible solutions to the primal and the dual problem, respectively. Then x and p are optimal if and only if

ui≡pi(aiTx−bi)=0for 1≤i≤m,vj≡(cj−pTAj)xj=0for 1≤j≤n.

Proof. Recall that, for primal feasible x and dual feasible p, we have ui≥0 and vj≥0 for all i and j. Moreover, cTx−pTb=∑iui+∑jvj.

(⇒) By the strong duality theorem, if x and p are optimal, then cTx=pTb, which implies that ui=vj=0 for all i and j. (⇐) follows from Corollary 4.2.

A. In standard form

We may assume that the rows of A are linearly independent. (See Example 4.3.)

The first complementary slackness condition is automatically satisfied by every feasible solution x to a primal problem in standard form, since Ax=b. Thus it gives no information on p.

However, if x is a nondegenerate optimal basic feasible solution, then the second complementary slackness conditions determine a unique solution to the dual problem, pT=cBTB−1, since xj>0 so that cj−pTAj=0 for all basic variables j∈B.

B. In general form

The first complementary slackness condition implies that pi=0 for all non-active constraints aiTx≥bi.

If x is a nondegenerate optimal basic feasible solution, then there are n linearly independent active constraints so that the rank of A is n. Therefore, a unique solution to the dual problem is determined by pTA=cT.

C. Degenerate case

On the other hand, if we are given a degenerate optimal basic feasible solution to the primal, then the complementary slackness may be of very little help in determining an optimal solution to the dual problem.

 

A geometric view

Consider the primal problem in general form. Let I⊆{1,…,m} such that (1) |I|=n and (2) ai for i∈I are linearly independent. Then the system aiTx=bi for i∈I has a unique solution xI​, which is a basic solution to the primal problem. We assume that xI is nondegenerate, that is, aiTx≠bi for all i∉I​.

Let p∈Rm be a dual vector (not necessarily dual feasible), and consider what is required for xI and p to be optimal solutions. We need:

  1. aiTxI≥bi for all i (primal feasibility),

  2. pi=0 for all i∉I (complementary slackness),

  3. pTA=cT​ or ATp=c (dual feasibility),

  4. p≥0 (dual feasibility).

From (2) and (3), we have ∑i∈Ipiai=c. It has a unique solution pI, since ai for i∈I are linearly independent. For pI to be dual feasible, we also need pI≥0n​.

Therefore, if (2) is satisfied, pI is feasible if and only if c is a nonnegative linear combination of ai for i∈I.

If x∗ is a degenerate basic solution to the primal, there can be several subsets I such that xI=x∗. Using different choices for I, and by solving the system ∑i∈Ipiai=c, we may obtain several dual basic solutions pI. It may then well be the case that some of them are dual feasible and some are not.

Still, if pI is dual feasible, i.e., pi≥0 for all i) and if x∗ is primal feasible, then they are both optimal, because we have been enforcing complementary slackness and Theorem 4.5 applies.

 

4.4 Optimal dual variables as marginal costs

Consider the primal problem in standard form. We assume the rows of A are linearly independent and that there is a nondegenerate basic feasible solution x∗​ which is optimal.

Let B be the corresponding basis matrix. Then xB∗=B−1b>0m by nondegeneracy. Now replace b by b+d for a sufficiently small perturbation vector d∈Rm so that B−1(b+d)>0m. This implies that the same basis B leads to a basic feasible solution of the perturbed problem as well.

Perturbing b has no effect on the reduced costs c¯=cT−cBTB−1A associated with the basis B. By the optimality of x∗ in the original problem, c¯≥0 and this establishes that the same basis is optimal for the perturbed problem as well. Thus, the optimal cost in the perturbed problem is

cBTB−1(b+d)=pT(b+d),

where pT=cBTB−1 is an optimal solution to the dual problem. Therefore, a small change of d in b results in a change of pTd in the optimal cost.

We conclude that each component pi of the optimal dual vector can be interpreted as the marginal cost (or shadow price) per unit increase of the i-th requirement bi.
